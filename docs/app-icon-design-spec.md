# Library App Icon — 全套设计方案

## 全局参数

### 画布与裁切

| 参数 | iOS | macOS |
|---|---|---|
| 源图尺寸 | 1024×1024 px | 1024×1024 px |
| 系统圆角 | 系统自动叠加（约 23.3% 圆角） | 需手动裁切（圆角半径 180px） |
| 安全区 | 内缩 64px（960×960 核心区） | 内缩 32px（960×960 核心区，因无系统自动裁切） |
| 导出 | 单张 1024×1024 PNG | 16/32/128/256/512 各 @1x @2x（共 10 张） |

### 调色板

```
AMBER           #C89B5A  (200, 155, 90)   — 主 accent
AMBER_LIGHT     #E6C38C  (230, 195, 140)  — 亮面/高光区
AMBER_DARK      #A07332  (160, 115, 50)   — 暗面/阴影区
AMBER_GLOW      #F1D2A0  (241, 210, 160)  — 辉光色

BG_NAVY         #16181E  (22, 24, 30)     — 深色背景（方案一、四）
BG_DARK         #111113  (17, 17, 19)     — 纯黑背景（方案三）
BG_CHARCOAL     #1C1C20  (28, 28, 32)     — 炭灰背景（方案五）
BG_CREAM        #F7F5F1  (247, 245, 241)  — 暖奶油背景（方案二）

PAGE_LEFT       #FCF9F2  (252, 249, 242)  — 左书页
PAGE_RIGHT      #F5F1E9  (245, 241, 233)  — 右书页（略暗，暗示光影）
PAGE_LINE       #D2CDC6  (210, 205, 198)  — 书页文字线
SPINE           #B9AF9E  (185, 175, 158)  — 书脊色
SHADOW          #000000  不透明度 8%-15%   — 投影
```

### 三态变体规则

| 变体 | 背景色 | 辉光强度 | 书页亮度 | 特殊处理 |
|---|---|---|---|---|
| **Light** | 各方案指定底色 | 100%（基准） | 100%（基准） | 无 |
| **Dark** | Light 底色再压暗 8-12% | 收敛至 65% | 降低至 85% | 整体对比度降低，避免夜间刺眼 |
| **Tinted** | 去色后叠加单色 | 去色 | 去色 | Light 版→灰度→覆盖 #C89B5A @ 31% 不透明度 |

| 元素 | Light | Dark | Tinted |
|---|---|---|---|
| 背景 | #16181E | #0E0E12 | 去色叠加 |

| 书页亮度 | 100% | 85% | 去色 |

- 在 1024px 源图上叠加圆角矩形遮罩（圆角半径 180px）
- 在遮罩内顶部区域叠加白色高光条：(22, 22) → (1002, 600)，圆角半径 178px，色 rgba(255,255,255,0.07)，模糊 10px
- 最终导出各尺寸时，边缘带 1px rgba(255,255,255,0.1) 的 inner stroke
