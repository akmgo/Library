# iOS Book Record Blueprint

## Positioning

阅读日记是一个只服务书籍的 iOS 记录应用。

它不阅读电子书，不计时，不做泛摘录，也不做复杂管理系统。

它也不设计目标。阅读分钟、页数进度、书摘和笔记只作为事实记录展示，不转化为任务、成就或达标压力。

## Core Objects

- Book
- ReadingLog
- BookText

## Main Flows

1. 添加一本书
2. 手动记录一次阅读时长
3. 保存一条书摘
4. 保存一条笔记
5. 在书籍详情中查看这本书的阅读记录、摘录和笔记

## Design Direction

- warm paper background
- calm solid cards
- native iOS navigation
- few controls
- strong typography
- no decorative animation

## Non-goals

- macOS
- widgets
- Live Activities
- timer
- reading goals
- streak targets
- achievement or task systems
- EPUB/PDF/TXT reader
- CloudKit
- generic snippets
- AI/export/calendar/backup features
